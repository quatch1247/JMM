JWT 토큰 발급 및 사용:

사용자가 로그인하면 서버는 JWT 토큰을 발급하고 클라이언트에 전달합니다.
클라이언트는 이후의 모든 API 요청에 이 JWT 토큰을 포함하여 서버에 전송합니다.
서버는 JWT 토큰을 검증하여 요청이 인증된 사용자로부터 온 것인지 확인합니다.
Refresh Token 발급 및 사용:

JWT 토큰과 함께 Refresh Token도 발급되어 클라이언트에 전달됩니다.
JWT 토큰이 만료되면 클라이언트는 Refresh Token을 사용하여 새로운 JWT 토큰을 요청합니다.
서버는 Refresh Token을 검증하고, 유효하다면 새로운 JWT 토큰을 발급합니다.
만약 Refresh Token이 유효하지 않거나 만료되었다면, 클라이언트는 다시 로그인해야 합니다.
